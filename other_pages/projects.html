Hash Table Analysis

This project involved creating multiple hash tables with different ways of handling collisions.
Once that was done, large sets of integer data were inserted into the tables and then the program recorded how much
time it took to perform each operation which measures how time efficient each hash table is.


Twitch Discord Bot

I created a simple Discord bot whose commands were to report the current
live Twitch streamer who had the most views, ping the server when a specified streamer went live,
and some other minor commands to mess around with friends. The goal of this project was to learn how
to use Twitch's API.


Dutch Bros. Drink Randomizer

My friend and I collaborated on the creation of a mobile application that
took the drink menu from Dutch Bros. and returned a random drink for the user if they were not
sure on what drink they wanted to order. You could also select specific drink types then choose a
random drink from those specific drink types.
